Examples - use of group additivity may be the best way to 
learn / start to use group additivity. 


Group Additivity is a powerful method to estimate accurate 
thermodynamic properties for many molecule and radical species.


Some of the nomenclature used includes:

*  represents double bond  C*C  is C=C 
 C represents Carbon

 #  represents triple bond  C#C  is C=C 

 j or .   is a radical site  CC.   is CH3C.H2

 O2 represents an organic acid, CCO2H is acetic acid

 OO represents a peroxide   CCOOH  is ethyl hydroperoxide 
                            COOC is dimethyl peroxide 
 M can represent Methyl   

In order for group additivity to work with halogens interaction groups 
are needed to account for the interaction of the halogens on adjacent 
carbon or other atoms. Interaction terms are only used for halogens on 
adjacent carbons, NOT on the same carbon. Interactions between Halogens 
on the same carbon are accounted for in the group value. 
 \Please see examples.

Halogen interaction groups exist for F, Cl, Br, and mixed halogens 


The Following Example file includes:

- Alkanes, 
- Alkenes
- Alkynes
- Alcohols
- Acids
- Aldehydes
- Ketones
- Amines
- Peroxides
- Hydroperoxides 
- Chlorines (halogens)
- Cyclics
- Aromatics
